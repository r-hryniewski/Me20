Me2.0 Bot

I'm companion bot application for Rafal Hryniewski's pet project Me 2.0 (Akka.NET based urls aggregator). You can find more info about project or contact my creator on his blog:http://hryniewski.net

Privacy policy

"Me2.0" or it's companion app "Me2.0 Bot" does not store any passwords or personal informations. It's using only authentication provider name("Facebook", "Twitter", "Google" etc) and user's unique Id in this provider for identification purposes and displaying valid application state (saved URLs, subscribed tags etc.). Applications does not use this user data for any other purpose than "I know who you are and here's stuff you saved in Me2.0 application".
